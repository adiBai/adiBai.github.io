## 演示

你可以点击[这里](http://henryhuang.github.io/test/)，查看演示。

## 安装

``` bash
$ git clone https://github.com/henryhuang/test.git themes/test
```
**test 需要 Hexo 2.7 及以上版本.**

## 启用

修改主题的设置文件`_config.yml`，把`theme`的值设置为`test`

## 更新

``` bash
cd themes/test
git pull
```
